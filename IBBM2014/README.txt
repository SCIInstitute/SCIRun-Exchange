Locations for data used by networks in this directory can be found in the networks themselves, as module notes.  Data will either be found in the SCIRunData directory (available at scirun.org) or from the IBBM data directory (available on the canvas website for the IBBM course).  For data in the SCIRunData file, make sure that the SCIRunData environment varible is set properly.  This can be set in the by clicking on the 'Configure' button on the bottom right of the SCIRun interface and then the 'Options' tab..
